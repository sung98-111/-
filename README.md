# -
الحضارات القديمة هي المصطلح الذي يُطلق على المجتمعات المعقدة والمستقرة التي ازدهرت في العالم القديم (آسيا، أفريقيا، وأوروبا) قبل آلاف السنين، تحديداً منذ حوالي 3000 سنة قبل الميلاد. نشأت معظم هذه الحضارات على ضفاف الأنهار الكبرى، مثل النيل والفرات، حيث وفرت المياه اللازمة للزراعة واستقرار السكان.
